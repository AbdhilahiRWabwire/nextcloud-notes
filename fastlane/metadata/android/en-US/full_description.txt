Companion app to Nextcloud Notes.

Features

    Multiple accounts
    List, create, edit, share, search and delete notes
    Manage categories
    Share text and links as new note into the app
    Mark notes as favorite
    Bulk delete
    Formatting with MarkDown (using RxMarkdown)
    Multi-Lingual User-Interface

Requirements

    Nextcloud¹ instance running
    Nextcloud Files app² enabled (> 3.9.0)
    Nextcloud Notes app³ enabled

¹ https://nextcloud.com/
² https://github.com/nextcloud/android
³ https://github.com/nextcloud/notes
